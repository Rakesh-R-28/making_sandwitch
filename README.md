# 🥪 Sandwich Maker (Python)

## 📌 Description

A simple Python function that creates a description of a sandwich based on given ingredients and options.

## 🚀 Features

* Choose bread type and filling
* Optional cheese selection
* Option to make the sandwich toasted
* Uses default values for flexibility


## 📖 Concepts Used

* Functions
* Default parameters
* Conditional statements
* String formatting (f-strings)

## ✅ Conclusion

This project demonstrates how to build flexible and reusable functions using optional parameters in Python.
